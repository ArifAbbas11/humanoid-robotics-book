# Physical AI & Humanoid Robotics Book

This repository contains a comprehensive guide on Physical AI & Humanoid Robotics: From Simulation to Embodied Intelligence. The book covers ROS 2, Gazebo/Unity simulation, NVIDIA Isaac, and Vision-Language-Action integration.

## Table of Contents

1. [ROS 2 Fundamentals](docs/ros-fundamentals/intro.md) - Middleware for robot control
2. [Simulation](docs/simulation/intro.md) - Physics simulation and environment building
3. [AI Navigation](docs/ai-navigation/intro.md) - Advanced perception and training
4. [Vision-Language-Action Integration](docs/vla-integration/intro.md) - Integrating LLMs and robotics

## Target Audience

Students, researchers, and hobbyists interested in embodied AI, robotics, and AI-human interaction.

## Technology Stack

- Docusaurus for documentation
- ROS 2 (Humble Hawksbill)
- Gazebo/Unity for simulation
- NVIDIA Isaac for AI
- Vision-Language-Action systems

## Getting Started

1. Install Node.js 18+
2. Clone the repository
3. Run `npm install`
4. Start the development server with `npm start`
